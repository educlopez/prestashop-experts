# Demo 3 - Creative

## Niche + target

Editorial / creative-agency layout. Uses the Owl carousel as hero, a QR code block in header, and Twitter-embedded timelines — the only demo that does. Feels like a lookbook more than a catalogue.

## Design characteristics

- Editorial rhythm: carousel > banners > featured > brands > newsletter.
- QR code in header — unusual detail, reinforces creative brand.
- Twitter timeline as social proof block.
- Fewer product-sliders; more narrative blocks.

## Use this demo when...

- Creative agency, photographer, or lookbook-style brand.
- Twitter/X presence is a major social channel.
- Client wants a 'less grid, more story' homepage.
