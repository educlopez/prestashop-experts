# Notificationr

1. This module can be used to display an notification to let visisters know that your site uses cookies. The module does not block prestashop and the theme using cookies. There are dozens of "Cookie law" module for prestashop, only one module blocks using cookies, other modules are just show a warnning.
2. This module can also be used to display floating banners.
