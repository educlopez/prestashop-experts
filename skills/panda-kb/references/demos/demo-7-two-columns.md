## Design characteristics

- Left sidebar dominates — filters and category tree always visible.
- Dense product-slider stack: categories + specials + featured.
- Facebook widget instead of newsletter for social retention.
- Classic e-commerce feel, highest comfort for returning shoppers.

## Use this demo when...

- Large catalogue that benefits from always-on category navigation.
- Client prefers classic grid e-commerce over minimal layouts.
- Facebook community is a primary channel.
