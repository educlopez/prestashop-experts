## Design characteristics

- Mirrored layout — everything reads right-to-left.
- Currency + language selectors promoted in header.
- Otherwise identical block palette to Demo 1/Demo 4.
- Tests Panda's bidi/CSS-logical-properties support.

## Use this demo when...

- Client targets Middle-Eastern / Hebrew-speaking markets.
- Multi-region store that needs RTL as a switchable mode.
- You need a proof that Panda supports RTL cleanly.
