# One click demo importer

1. IMPORTANT Upload the "modules" and "stupload" folders in the "One click demo import" folder, which you can find in the main.zip file you have downloaded from www.sunnytoo.com, into the root folder of your Prestashop installation.
2. Click "Demo x" buttons in the Theme editor module to import predefined demos. You have two choises, one is importing all data, another is improting colors only.
3. "Featured categories slider" module and "Product slider for each category" module can not be expored/imported, because of categories are differnt from sites to sites.
4. Sample accounts will be appied to these modules "Facebook page plugin", "Twitter Embedded Timelines" and "Instagram block", so you are going to use your account instead after importing.
