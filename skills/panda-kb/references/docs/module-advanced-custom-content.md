# Advanced custom content

We added a "Advanced content editor" feature to the module, with it you can easily create elements like FAQs, Tabs, Responsive blocks, Icon with text blocks, Text with image blocks, Google maps, etc. The content of element pages (Actually they are CMS pages) on the main demo were created by using the feature.

This gif shows how the feature works, you can use it to create a responsive block and put the block on the homepage or a CMS page or some other places.
