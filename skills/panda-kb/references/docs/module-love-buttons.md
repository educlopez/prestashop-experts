# Love buttons

1. If a customer loves a product from homepage and the category page, for cache and loading speed reasons we can not tell if the customer has already loved the product or not, so if the customer loved the product before, then the first click will be disregard. If the customer did not love it before, then the first takes effect. If the customer clicks the same love button again, then the second click always means unloving.
