- Compact 980px layout — denser on small/medium screens.
- Tag cloud near footer, SEO-leaning.
- Notification bar pinned for promotions.
- QR code block as footer utility.

## Use this demo when...

- Audience is mostly on older laptops or dense desktop users.
- Retro/utility brands that want a 'forum-era' feel.
- Client wants a narrower reading column.
