# Stickers block

The module gives you full control over native stickers and custom stickers, you can decide displaying native sticks or not.

1. There are two types "Text sticker" and "Image sticker". If you have upload an image to a sticker, then the stiker becomes a "Image sticker".
2. PrestaShop shows "New", "Sale", "Online only" out by default, if you do not want them, just delete or disable them from lists, no need to remove them product by product.
3. Stickers do not show on the main image on the product page, because of they can not work fine with the zoom feature.
