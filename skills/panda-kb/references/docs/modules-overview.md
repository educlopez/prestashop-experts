# Panda 2 modules


1. The mobile header bottom hook is a little bit special, links in it would be streched, like this.
	![Mobiel header bottom](assets/panda/mobile_header_bottom.jpg)
2. Search bar mod displays as a button by default, if you want to have a search box on the mobile heard, go to the Search bar mod, set the "How to display search box on mobile header" to "An input file".
	![Search box on mobile header](assets/panda/search_box_on_mobile_header.jpg)
