## Niche + target

Megamenu rendered vertically as a sidebar instead of a horizontal bar. Suits large category trees (electronics, supermarkets, hypermarket-style catalogues). Includes the full product + blog + brand module stack.

## Design characteristics

- Vertical megamenu pinned left — reminiscent of marketplace/hypermarket sites.
- Hero swiper + Owl carousel coexist — double carousel.
- Special slider + homenew + featured chain.
- Full content layer (blog + featured articles + brands).

## Use this demo when...

- Large category tree that doesn't fit a horizontal nav (50+ root categories).
- Electronics, supermarket, multi-brand marketplace.
- Client has seen Amazon/eBay-style menus and wants similar.
