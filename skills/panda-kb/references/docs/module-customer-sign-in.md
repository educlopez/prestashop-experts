# Customer "Sign in" link mod

The module has a quick login feature which looks like the image below, but you may get this message "This connection is not secure. Logins entered here could be compromised".

To get rid of it, you need to enable SSL. More info here: https://support.mozilla.org/en-US/kb/insecure-password-warning-firefox?as=u&utm\_source=inproduct
