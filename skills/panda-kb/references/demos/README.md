# demos/

Fichas de las 19 demos oficiales de Panda. Una por demo.

**Profundidad de ingesta**: homepage + category + 1 product page por demo.

**Frontmatter requerido:**
```yaml
---
demo_name: Demo 1 - Main
niche: fashion
layout_width: 1440
preview_url: https://panda2.sunnytoo.com/demo1/
modules_used: [stickyhead, megamenu, stheromain, stproductsslider, stsocialfeed]
pages_ingested: [home, category, product]
ingested: 2026-04-18
ps_version: [8, 9]
tags: [panda, demo]
---
```

Se usa para entender **qué módulos combinan en qué niches** y servir de referencia al agente cuando proponga layouts.
